3. Description of the program.
   - Запуск игры и стартовое меню:
       > Игра будет запускаться с помощью простого скрипта на Python, который инициализирует PyGame и загружает необходимые ресурсы (изображения, звуки и т.д.).  
         Стартовое меню будет оформлено в минималистичном стиле, с яркими цветами и простой графикой. На экране будут следующие элементы:    
       > - Логотип игры в верхней части экрана. 
       > - Кнопка "Начать игру", которая запускает игровой процесс. 
       > - Кнопка "Настройки", позволяющая пользователю настроить звук и управление. 
       > - Кнопка "Выход", закрывающая игру. Информация о разработчиках внизу экрана.
   - Завершение игры и финальное окно:
       > Игра будет завершаться, когда игрок столкнется с препятствием или упадет на землю.  
         В финальном окне будет отображаться:  
       > - Результат игрока (количество пройденных препятствий). 
       > - Таблица рекордов с возможностью сохранения лучших результатов. 
       > - Кнопка "Начать заново", позволяющая начать новую игру. 
       > - Кнопка "Выход", чтобы вернуться в стартовое меню. 
       > - Возможность сохранить игру (например, если игрок хочет продолжить позже).  
   - Действия пользователя с игровым персонажем:
       > Игрок управляет персонажем, который представляет собой птицу. Доступные действия:  
       > - Перемещение вверх при нажатии клавиши (например, пробел или стрелка вверх). 
       > - Падение под действием гравитации, если клавиша не нажата. 
       > - Игрок может собирать бонусы, которые будут появляться в игре (например, монеты, звездочки, или дополнительные жизни).
   - Взаимодействия между игровыми персонажами:
       > В игре будет несколько препятствий, с которыми игрок должен избегать столкновения. Возможные взаимодействия:  
       > - Столкновение с препятствиями, что приводит к завершению игры. 
       > - Игрок может собирать бонусы, которые будут появляться между препятствиями. 
       > - Если игрок соберет определенное количество бонусов, он может получить временные улучшения.  
           Например: 1. Временную неуязвимость 2. Дополнительную жизнь 3. Множитель очков
   - Звуковое сопровождение:
       > Во время игры будет звучать фоновая музыка, создающая атмосферу. Звуковые эффекты будут включать:  
       > - Звук при нажатии клавиши для прыжка. 
       > - Звук при столкновении с препятствиями. 
       > - Звук при сборе бонусов. 
       > - Звук при завершении игры (например, печальный звук).
   - Премиальные качества игры:
       > - Уникальный стиль графики: Яркие и красочные изображения, которые привлекают внимание. 
       > - Интуитивно понятное управление: Простое управление, доступное для игроков всех возрастов. 
       > - Разнообразие уровней сложности: Постепенное увеличение сложности, позволяющее удерживать интерес игроков. 
       > - Система достижений: Награды за выполнение определенных задач (например, пройти 10, 20, 50 препятствий). 

4. Project Description. Описание структуры проекта.
    - Структура проекта: 
        > - Проект будет состоять из нескольких ключевых компонентов, каждый из которых будет отвечать за свою часть функциональности. Ниже приведены основные элементы структуры проекта:

    - Главный скрипт:
        > - Инициализация PyGame и загрузка необходимых ресурсов (изображения, звуки).
        > - Запуск стартового меню и управление игровым процессом.
    
    - Стартовое меню:
       > - Отображение логотипа игры.
       > - Кнопки для начала игры, настроек и выхода.
       > - Обработка событий нажатия кнопок.
       > - Настройка громкости звуков и сложности игры.

    - Игровой процесс:
       > - Основной игровой цикл, который управляет состоянием игры.
       > - Управление персонажем (птицей) и обработка ввода от пользователя.
       > - Генерация препятствий и бонусов, а также их движение.
       > - Обработка столкновений между персонажем, препятствиями и бонусами.

    - Финальное окно:
       > - Отображение результатов игрока и таблицы рекордов.
       > - Кнопки для начала новой игры и выхода в меню.
       > - Возможность сохранения текущего состояния игры.
    
    - Звуковое сопровождение:
       > - Фоновая музыка, создающая атмосферу.
       > - Звуковые эффекты для действий игрока (прыжок, столкновение, сбор бонусов).
       > - Возможные схемы.
       > - Звуковые эффекты для нажатия кнопок ("клик", "тыц" и т.д.).

    - Схема взаимодействия элементов кода:
       > - Главный скрипт → Стартовое меню → Игровой процесс → Финальное окно.
       > - Игровой процесс взаимодействует с классами для управления персонажем и препятствиями.
       > - Функция для запуска всего проекта
  
    - Схема задействованных окон:
       > - Стартовое меню → Главное окно (игровое) → Финальное окно.
    
    - Схема классов:
       > - Game: основной класс, управляющий игровым процессом.
       > - Bird: класс для управления персонажем.
       > - Obstacle: класс для создания препятствий.
       > - Bonus: класс для создания бонусов.
       > - Menu: класс для управления стартовым меню и финальным окном.
    
    - База данных
       > - Для хранения рекордов можно использовать простую текстовую базу данных или JSON-файл, который будет обновляться при каждом новом рекорде.
       > - В отдельные текстовые файлы может сохраняться прогресс пользователя в конкретной игре (кол-во собранных бонусов, время в игре и т.д.)

5. Program code plan. Описание кода.
Переменные и константы
    - Переменные:
       > - score: текущий счет игрока.
       > - high_scores: список рекордов.
       > - is_game_over: флаг завершения игры.
    - Константы:
       > - SCREEN_WIDTH, SCREEN_HEIGHT: размеры игрового окна.
       > - BIRD_GRAVITY: сила гравитации для птицы.
       > - OBSTACLE_SPEED: скорость движения препятствий.
    - Функции:
       > - init_game(): инициализация игры и загрузка ресурсов.
       > - draw_menu(): отрисовка стартового меню.
       > - start_game(): запуск игрового процесса.
       > - update_game(): обновление состояния игры (движение птицы, препятствий).
       > - check_collisions(): проверка столкновений.
       > - show_results(): отображение финального окна с результатами.
    - Классы:
       > - Game: основной класс, управляющий игровым процессом.
       > - Bird: класс для управления персонажем (птицей).
       > - Obstacle: класс для создания и управления препятствиями.
       > - Bonus: класс для создания и управления бонусами.
       > - Menu: класс для управления меню и финальным окном.
    - Библиотеки:
       > - pygame: основная библиотека для создания игры.
       > - random: для генерации случайных препятствий и бонусов.
       > - json: для работы с файлами рекордов.
